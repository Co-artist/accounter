## 问题定位
用户反馈“气泡的位置再往下一点点，上边和登录框靠得太近了”。
虽然用户说“和登录框靠得太近”，但根据布局，Logo 是在登录框下方的，所以气泡是在 Logo 的上方，可能离上面的登录框底部边缘太近了，或者用户觉得气泡太高了。
当前样式：
```css
.speech-bubble {
  top: -45px;
  right: -80px;
  /* ... */
}
```
气泡的高度大约是 `20-30px` 加上 padding，`top: -45px` 意味着气泡底部距离 Logo 顶部还有一段距离。

## 修复方案
将气泡向下移动，即增大 `top` 的值（使其不那么负）。
例如改为 `top: -35px` 或 `top: -30px`，让气泡离 Logo 更近一些，从而远离上方的登录框。

## 实施步骤
修改 `src/components/InteractiveLogo.vue` 中的 `.speech-bubble` 样式。

## 预期效果
气泡会向下移动，与 Logo 的距离更紧凑，从而缓解与上方元素的拥挤感。
