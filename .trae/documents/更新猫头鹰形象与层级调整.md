## 任务拆解
1.  **重绘猫头鹰 SVG 形象**：
    *   根据用户提供的 Logo 描述（黄色圆形身体、大眼睛、含着钱币、小翅膀），重新编写 `OwlMascot.vue` 中的 SVG 代码。
    *   **身体**：黄色大圆。
    *   **耳朵**：头顶两个小三角形/角。
    *   **眼睛**：两个大白圆，内含黑色瞳孔和白色高光。
    *   **嘴巴**：倒三角，含着一个黄色小圆（钱币），钱币内有 "¥" 符号。
    *   **翅膀**：为了实现“遮眼”动效，需要将翅膀设计得更灵活。
        *   **常态**：翅膀在身体两侧，呈自然收拢状。
        *   **遮眼态**：翅膀需要旋转并伸长，或者移动到眼睛前方。考虑到扁平风格，可以设计为“旋转上来遮挡”。
2.  **调整层级 (Z-Index)**：
    *   用户要求猫头鹰图层在登录框之下。
    *   当前 `OwlMascot` 在 `LoginPage.vue` 的 `.auth-form` 内部。
    *   **方案**：将 `OwlMascot` 移出 `.auth-form`，放在 `.auth-form` 的兄弟节点或父级容器中，并设置 `.auth-form` 的 `z-index` 高于 `OwlMascot`。
    *   考虑到定位方便，可以将 `OwlMascot` 放在 `.auth-form` 之前，或者使用绝对定位相对于 `.auth-page` 或一个新的包装器。
    *   **最佳实践**：在 `.auth-form` 外部包裹一个 `div.login-container`，设为 `position: relative`。将 `OwlMascot` 和 `.auth-form` 都放在里面。`OwlMascot` 绝对定位到右上角，`z-index: 0`；`.auth-form` 相对定位，`z-index: 1`。

## 具体实施计划

1.  **修改 `src/views/LoginPage.vue`**：
    *   调整 DOM 结构：
        ```html
        <div class="login-wrapper" style="position: relative;">
          <OwlMascot ref="mascot" /> <!-- 移到 form 外面 -->
          <form class="auth-form" ...> ... </form>
        </div>
        ```
    *   CSS 调整：确保 `.auth-form` 有背景色且 `z-index` 高于猫头鹰。调整猫头鹰的 `top` 和 `right` 定位，使其看起来像是从登录框后面探出头/趴在上面。

2.  **重写 `src/components/OwlMascot.vue`**：
    *   **SVG 更新**：
        *   ViewBox: `0 0 200 200`
        *   Body: `<circle cx="100" cy="100" r="90" fill="#FFD54F" stroke="#333" stroke-width="3" />`
        *   Ears: `<path d="..." fill="#FFD54F" stroke="#333" ... />`
        *   Wings (Left/Right): 设计为椭圆或圆弧状，初始位置在身体两侧下方。关键点是旋转中心 (`transform-origin`) 要设置在翅膀根部，以便旋转上来遮眼。
        *   Eyes: `<g class="eyes"> ... </g>` (白底黑瞳)
        *   Beak & Coin: `<path ... />` (嘴) + `<circle ... />` (钱币) + `<text>¥</text>`
    *   **动画更新 (GSAP)**：
        *   `startIdle()`: 翅膀微动，身体微晃，眼睛眨动。
        *   `coverEyes()`: 左右翅膀旋转 (`rotation`) 并位移 (`x`, `y`) 到眼睛位置。可能需要配合 `scale` 伸长翅膀。
        *   `uncoverEyes()`: 恢复原状。

## 预期效果
*   猫头鹰形象更新为黄色圆脸、含钱币的 Logo 样式。
*   猫头鹰位于登录框右上角的后方（部分被遮挡或趴在边缘）。
*   输入密码时，翅膀会举起来遮住大眼睛。
