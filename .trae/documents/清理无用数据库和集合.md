## 清理无用数据库和集合

### 1. 分析当前数据库状态
- 当前项目使用的数据库：`acconuter`
- 无用的数据库：`accounting-app`、`accounting_app`
- 系统数据库：`admin`、`config`、`local`（保留）

### 2. 清理步骤

**步骤1：运行数据库清理脚本**
- 执行 `cleanup-databases.js` 脚本
- 删除无用的数据库：`accounting-app`、`accounting_app`
- 保留系统数据库和当前项目使用的数据库

**步骤2：验证清理结果**
- 检查剩余数据库列表
- 确认只保留了 `acconuter` 和系统数据库

**步骤3：检查当前项目数据库集合**
- 确认 `acconuter` 数据库中只有 `users`、`transactions`、`categories`、`budgets` 集合

### 3. 执行命令

```powershell
# 运行清理脚本
node cleanup-databases.js

# 验证清理结果
node verify-db.js
```

### 4. 预期结果
- ✅ 成功删除 `accounting-app` 数据库
- ✅ 成功删除 `accounting_app` 数据库
- ✅ 保留 `acconuter` 数据库
- ✅ 保留系统数据库（`admin`、`config`、`local`）
- ✅ 确认当前项目数据库正常工作

### 5. 注意事项
- 确保当前项目数据库 `acconuter` 正在使用中
- 清理操作不可逆，请谨慎执行
- 建议在执行前备份重要数据

通过以上步骤，我们将成功清理无用的数据库，只保留当前项目使用的数据库和必要的系统数据库。
